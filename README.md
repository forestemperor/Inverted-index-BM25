# Inverted-index+BM25

本项目手动实现了Elasticsearch等搜索引擎项目的倒排索引与BM25这2个主要的功能。对于小型项目尤其是自己搭建搜索引擎这方面，不再需要繁琐的安装Elasticsearch，而直接使用其对于功能完成搜索。
